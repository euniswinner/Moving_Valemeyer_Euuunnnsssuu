# Moving_Valemeyer_Euuunnnsssuu
##Moving valemeyer##
A study of how a town moved to escape flooding and what it can teach us about moving towns away from wildfires or rising seas today.


In 1993, a flood destroyed the twon of Valmeyer, Illinois. Instead of rebuilding in the same place, the whole twon moved 2 miles away to higher ground. Why did this work? Can other towns use this as a model today?
